# NestJS + React app

Back and Front should be running in parallel.

## How to run
1. Install all libraries both for backend and frontend.
1. Add `ormconfig.json` with all details for backend.
1. Run both backend and frontend.

If everything done correctly, two sections will be available to you:
1. /Admin - for the main dashboard view.
1. /Login - the example of login page.
You can import entries from the table on dashboard page by clicking "sync entry" button if the `ormconfig.json` set up correctly.
